# KiepCommunication
### The first communication tools we created for Kiep
 
Features of KiepCommunication:
 - C# Windows Forms application
 - Using 1 button scanning input method
 - Control with numpad 
 - Text to speech
 - Autocompletion
 - Speak "yes" and "no" .WAV files via numpad
