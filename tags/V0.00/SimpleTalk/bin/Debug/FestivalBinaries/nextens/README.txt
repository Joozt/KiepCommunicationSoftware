SUMMARY
----------------------------------------------------------------------

This the README file of the binary distribution of NEXTENS and
NINTENS. NEXTENS stands for 'Nederlandse Extensie voor
Tekst-naar-Spraak' (Dutch extension for text-to-speech). It is an
extension to the Festival speech synthesis system providing text to
speech conversion for Dutch. NINTENS stands for 'Nederlandse Interface
voor Tekst-naar-Spraak' (Dutch Interface for text-to-speech). It is a
graphical user interface on top of NEXTENS.  For more information, see
http://nextens.uvt.nl.

--------------------------------------------------------------------------------
INSTALLATION
--------------------------------------------------------------------------------

Windows
-------

Extract the files from the 'nextens-win-bin-1.0-beta.zip' archive in
the directory of your choice, for instance, in 'c:/nextens'. This will
create a directory called 'nextens-win-bin-1.0-beta'.

Note: some combinations of Cygwin and Window versions have trouble
with directory names that contain spaces (e.g. "c:\Program
Files\"). If you run into this, try installing in a directory path
without spaces.


----------------------------------------------------------------------
USAGE
----------------------------------------------------------------------

Windows
-------

To start NEXTENS, double click on the batch file called 'nextens.bat'
in the directory 'nextens-win-bin-1.0-beta'.

To start NINTENS, double click on the batch file called 'nintens.bat'
in the directory 'nextens-win-bin-1.0-beta'.
